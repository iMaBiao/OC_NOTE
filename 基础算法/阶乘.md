#### 阶乘

这个实现很简单，就是递归的基本原理。还有著名的裴波那切数列，都是这一类问题。

（1）退出条件：参数为0或者1的时候，返回1  

（2）递归：n * f(n-1)

##### 如果要做好一点，就是判断一下参数，不要太大，否则程序会傻掉的（数值越界）

```
int factorial(int n) {
    if (n > 100) {
        return -1; // 太大了，算不出来，会越界
    }
    if (n == 1 || n ==0 ) {
        return 1;
    }
    return n * factorial(n - 1);
}
```
