#### 冒泡排序

http://www.cnblogs.com/kkun/archive/2011/11/23/bubble_sort.html

- 基本思想是找出最小的一个，放好；然后往前走一步，在剩下的里面找出最小的一个，放好；再往前走一步；===一直走到最后一步；

- 实现也简单，i，j两层循环嵌套就可以了。

```
void bubble_sort(int s[], int length) {
    for (int i = 0; i < length; i++) {
        for (int j = i; j < length; j++) {
            if (s[i] > s[j]) {
                int temp = s[i];
                s[i] = s[j];
                s[j] = temp;
            }
        }
    }
}
```
